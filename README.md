# Student Portal

## Overview
The **Student Portal** is a web-based application designed for students to view their personal profile, class timetable, attendance records, and course information. The portal is **responsive**, clean, and simple to navigate, suitable for laptops and mobile devices.

---

## Features
- **Login & Registration**
  - Secure login system with simple validation.
  - Registration page for new users (demo only, not connected to a database).

- **Dashboard**
  - Overview of total courses, attendance rate, and GPA.
  - Attendance chart using Chart.js.
  - Announcements and images for events or notices.

- **Courses**
  - Displays a list of enrolled courses with course code, name, lecturer, and credit hours.
  - Clean, responsive table layout.

- **Attendance**
  - Shows attendance records per course.
  - Responsive table design.
  - Includes total classes attended and attendance percentage.

- **Profile**
  - Displays student personal details including ID, semester, campus, phone, and address.
  - Includes a **timetable and class locations**.
  - Responsive layout.

- **Navigation**
  - Navbar with working links to all pages.
  - Responsive burger menu for mobile devices.

- **Footer**
  - Stays at the bottom of the page using flexbox.
  - Displays copyright information.

---

- **HTML5 & CSS3**
- **JavaScript** for interactivity and simple login validation
- **Bootstrap 5** for responsive layout and components
- **Chart.js** for attendance visualization

## Notes
- The login system is currently client-side only and **not connected to a backend database**.  
- The portal is **fully responsive** using Bootstrap, suitable for desktops, laptops, and mobile devices.  
- The timetable, attendance records, and courses are **static demo data**.
